# Contract-Testing

#### 1. Clone repo and install npm packages

```bash
git clone https://github.com/Dev-Elie/Contract-Testing.git
cd Contract-Testing
npm install
```
#### 2. Start provider

`npm start`

#### 3. Run tests

```bash
npm run test:consumer
npm run test:provider
```
// "test": "jest tests/consumer.spec.js && jest tests/provider.spec.js",
